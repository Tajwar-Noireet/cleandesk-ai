# CleanDesk AI

CleanDesk AI is an intelligent assistant for keeping your workspace organized and efficient.

## Project Structure

The project is organized as follows:

```
cleandesk-ai/
  ├── frontend/       # User interface and client-side application
  ├── backend/        # Server API, database management, and business logic
  ├── ai-service/     # AI/ML services and model integration
  ├── docs/           # Project documentation and specifications
  └── README.md       # Project overview
```
